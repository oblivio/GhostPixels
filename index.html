Ghost Pixels | About

Ghost Pixels is a JavaScript steganography tool that allows users to hide a message into the pixels of a random image.

More than just hiding the data, you can encrypt the data with your own password. Without the password, the data inside the pixels is worthless.

GhostPixels is open-source software released under GPLV3. Developed by Fabian Valle. Source code available on Github.
